Run C4Engine.exe in the C4 Engine folder to play.


Controls:

WASD- move/strafe
Mouse- Look
Left Mouse Button- Attack
1- Open Inventory
Space- Jump
~- Open Command Line


Commands:

"pickup #" (from 1-12)- spawns a pickup based on item #
"bot"- spawns and enemy AI
"godmode"- adds all items to inventory
"disablebots"- removes all enemy AI
"enablebots"- resumes spawning of AI


Original level was too big for github so the debugging level is used instead
